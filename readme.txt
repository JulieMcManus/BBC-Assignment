Python 3.6 must be installed and can be downloaded from here:-
https://www.python.org/downloads/release/python-360/

selenium has to be installed as follows:-
from the command line - 
pip install -U selenium

Create a folder C:\Chromedriver
download chromedriver from https://sites.google.com/a/chromium.org/chromedriver/home
extract to C:\Chromedriver

edit user variables PATH to add C:\Chromedriver\chromedriver_win32\

BBC iPlayer radionav test:-
Download all scripts to the same folder and run BBCExcercise.py
